Grid System

O CSS Grid Layout é um sistema de layout bidimensional para a web que permite criar layouts complexos de forma simples e eficiente. Ele divide o espaço disponível em linhas e colunas, permitindo que os elementos sejam posicionados em qualquer lugar dentro dessa grade.

Exemplos

Com grid

.container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; }

Sem grid

.container { display: block; }

Flex System

O Flexbox (Flexible Box Layout) é um modelo de layout unidimensional que facilita a distribuição de espaço entre os itens em um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico. Ele é especialmente útil para layouts responsivos e alinhamento de itens em uma única direção, seja em linha (horizontal) ou em coluna (vertical).

Exemplos

Com flex

.container { display: flex; justify-content: space-between; }

Sem flex

.container { display: block; }
